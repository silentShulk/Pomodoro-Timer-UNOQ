# Pomododo Timer on LCD

## Modules
- Arduino UNO Q
- 4x4 Membrane Keypad
- Joystick (KY-023)
- 1602 LCD
- BreadBoard
- Resistors and Cables

## Setup                                                                        
Wall Charger -- (lightning port) Hub -- (usbc port) Arduino UNO Q 
                                                                                  


### Description
A pomodoro timer displayed on an LCD that can be controlled with a joystick a keypad
